## 3. Platform-by-platform setup checklist

### Facebook Business Page
- **Category:** Clothing (Brand) or Clothing Store
- **About section:** use MID description above
- **CTA button:** "Shop Now" (once site is live) or "Message Us" (for now)
- **Cover photo size:** 820 x 312 px (desktop), keep key text centered for mobile crop safety
- **Profile picture size:** 170 x 170 px (displays as circle — keep logo mark centered, avoid tight text at edges)

### Instagram
- **Bio:** use SHORT description (150-character limit)
- **Category:** Clothing (Brand)
- **Link:** use a Linktree/bio-link if multiple destinations (shop, WhatsApp, latest drop) are needed
- **Profile picture:** 320 x 320 px minimum, displays circular
- **Highlight cover suggestions:** Shop, Reviews, Behind the Scenes, Bulk Orders

### LinkedIn (Personal profile — Dipto/founder)
- **Headline:** "Founder, CodeWear | Developer-First Apparel Brand"
- **Featured section:** add CodeWear as a featured project with a link/description

### LinkedIn (Company Page)
- **Tagline:** Debug Your Day
- **Industry:** Apparel & Fashion (or Retail)
- **Company size:** 1–10 employees (adjust as accurate)
- **About section:** use LONG description above
- **Cover image size:** 1128 x 191 px
- **Logo size:** 300 x 300 px

### YouTube
- **Channel name:** CodeWear
- **Channel description (About tab):** use LONG description above
- **Keywords/tags for channel settings:** programmer t-shirt, developer merch, coding humor, tech apparel, coder gifts, dev culture
- **Banner size:** 2560 x 1440 px (safe area for text/logo: 1546 x 423 px, centered)
- **Profile picture:** 800 x 800 px

### Google Business Profile
- **Category:** Clothing store (primary), Online shop (secondary if applicable)
- **Short description:** use SHORT description
- **From the business (long description):** use LONG description above
- **Attributes to fill in:** online appointments (N/A), online ordering, delivery options
- **Photos needed:** logo, cover photo, at least 3–5 product photos, and (if applicable) a "team/behind the scenes" photo — profiles with more photos rank and convert better

### WhatsApp Business (worth setting up if not already)
- **Business description:** use SHORT description
- **Category:** Clothing store
- **Catalog:** upload product photos with prices directly — very commonly used for D2C brands in Bangladesh

## 4. Visual identity reference (for consistency across all platforms)

- **Primary colors:** Deep charcoal/navy (`#0D1B2A`) + terminal green accent (`#00C896` or classic terminal green `#39FF14` for higher contrast)
- **Typography feel:** Monospace or code-editor-inspired font for headings (e.g. JetBrains Mono, Fira Code, Space Mono), clean sans-serif for body text (e.g. Inter, Poppins)
- **Visual motifs:** terminal windows, code syntax highlighting colors, `<>` brackets, git/commit iconography, minimalist line-art of dev tools (keyboards, brackets, cursors)
- **Tone of voice:** witty, in-the-know, slightly self-deprecating dev humor — never corporate-sounding

## 6. Info you'll be asked for when opening any of these accounts

Have these ready before starting setup on any platform:

- Business/brand legal or trade name (if registered) and the DBA name "CodeWear"
- Business category (see per-platform notes above)
- Business email address (dedicated, not personal — e.g. hello@codewear.com or a Gmail like codewear.bd@gmail.com)
- Business phone number (WhatsApp Business number works across most platforms)
- Physical address or service area (Dhaka, Bangladesh — can mark as "delivers to customer" if no storefront)
- Website URL (once live) — until then, can link to Facebook Page or Instagram as the primary "website"
- Logo file (square, min 500x500px, PNG with transparent background)
- Cover/banner images (sized per platform above)
- 3–5 product photos minimum
- Payment/order methods to list (bKash, Nagad, COD, bank transfer — common for BD D2C brands)
